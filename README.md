# Rodavel

Open-source infrastructure modules for AWS. Drop-in, serverless, typed end-to-end, free to self-host.

Each module does one thing well, runs on your AWS account, and is MIT licensed. Pay only if you'd rather not manage it yourself.

Find out more on [Rodavel.com](https://rodavel.com)
